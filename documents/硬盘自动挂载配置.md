## 建议挂载位置

### 方案1: 挂载到 /media/binzhan/tools
Ubuntu "Other Locations" 会显示 `/media/用户名/` 下的挂载点。

```bash
sudo umount /mnt/tool
sudo mkdir -p /media/binzhan/tools
sudo mount /dev/sdb2 /media/binzhan/tools
```

### 方案2: 挂载到 /home/binzhan/tools
直接放在用户目录下，方便访问。

```bash
sudo umount /mnt/tool
sudo mkdir -p /home/binzhan/tools
sudo mount /dev/sdb2 /home/binzhan/tools
```

## 自动挂载配置 (fstab)

如果需要开机自动挂载，可以在 `/etc/fstab` 中添加：

```
# /dev/sdb2 - Tool盘 (NTFS)
UUID=10D6980DD697F168  /media/binzhan/tools  ntfs  defaults,uid=1000,gid=1000,umask=000  0  0
```

或者使用 /mnt/tool：
```
UUID=10D6980DD697F168  /mnt/tool  ntfs  defaults,uid=1000,gid=1000,umask=000  0  0
```
