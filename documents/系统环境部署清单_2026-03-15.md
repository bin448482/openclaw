## 重装系统参考

### 基础环境准备
```bash
# 1. 确认硬盘挂载 (已有fstab配置)
# /etc/fstab 包含:
UUID=10D6980DD697F168 /mnt/tool ntfs defaults,nls=utf8,uid=1000,gid=1000,dmask=022,fmask=133 0 0
```

### 应用安装顺序

```bash
# 1. 安装 Samba (独立)
sudo apt install samba

# 2. 安装 CasaOS (一键脚本)
curl -sSL https://get.casaos.io | sudo bash

# 3. 安装 sing-box
curl -fsSL https://sing-box.app/deb-install.sh | sudo bash

# 4. 配置数据盘 Samba 共享 (编辑 /etc/samba/smb.conf)
# 添加:
[tool]
   path = /mnt/tool
   browsable = yes
   writable = yes
   guest ok = yes
   force user = binzhan
   force group = binzhan

# 5. Podman (通常预装) 运行 Nextcloud
podman run -d --name nextcloud -p 8080:80 -v /var/www/html/data nextcloud:latest

# 6. CoPaw (手动安装，需回忆安装命令)
# curl -fsSL <URL> | bash
```
