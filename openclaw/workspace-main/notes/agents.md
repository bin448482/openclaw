# notes/agents.md — 笔记文件命名规范

## 标题格式

每个笔记文件的标题必须包含时间戳，精确到分钟，格式如下：

```
YYYY-MM-DD HH:mm-标题内容
```

- 日期与时间之间有一个空格
- 时间使用 24 小时制，格式为 HH:mm
- 时间戳与标题之间用 `-` 分隔
- 标题应简洁，准确概括核心内容

## 示例

```
2026-03-23 08:42-AI与人的信息单向输出问题.md
2026-03-22 14:30-关于项目架构的反思.md
```

## 其他规范

- 文件使用 `.md` 扩展名
- 文件名中避免使用特殊字符
- 标题反映核心观点，而非泛泛而谈
